# dG9kZGxl - BACKEND TASK

## Getting Started

### Installation

To install all dependencies, run:

```bash
npm install
```

### Running Project

To start the project, run:

```bash
node_modules/.bin/sequelize db:migrate
npm start
```

### Running Unit Tests

To start the project, run:

```bash
npm test
```
